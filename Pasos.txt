1. Tener la carpeta local del proyecto
2. Ingresar a la cuenta de GitHub
3. Crear un repositorio par el proyecto
4. Abrir la carpeta local en Visual Code
5. Abrir una terminal
6. Inicializar repositorio: git init
7. Vincular repositorio local con el remoto:
git remote add origin https://github.com/josemendozaloor/gestionsoft.git
8. Ver status: git status
9. agregar un archivo al commit: git add inicio2.html
10. agregar todos los archivos al commit: git add .
11. Ver status: git status
12. Establecer configuracion global correo:
git config --global user.email "jose.mendoza.loor@utelvt.edu.ec"
13. Establecer configuracion global usuario
git config --global user.name "josemendozaloor"
14. agregar un mensaje al commit: 
git commit -m "agregue el archivo inicio2.html"
15. Ver status antes de subir: git status
16. Subir todo a Github: git push
17. La primera vez sitiar la origin master:
git push --set-upstream origin master
18. Revisar los cambios en la pagina de GitHub
19. Modificar archivos o carpetas del proyecto
20. Ver status: git status
21. Agregar todos los archivos cambiados: git add .
22. Ver status: git status
23. agregar un mensaje al commit: 
git commit -m "agregue el archivo pasos.txt"
24. Subir todo a Github: git push
25. Revisar los cambios en la pagina de GitHub


Crear un nuevo repositorio en la línea de comando
git init 
git add README.md 
git commit -m "primer compromiso" 
git rama -M principal 
git push - u origen principal


Enviar un repositorio existente desde la línea de comando
git remoto agregar origen https://github.com/josemendozaloor/gestionsoft.git
git rama -M principal 
git push -u origen principal
